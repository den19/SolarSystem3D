Инструкции
Следуя видеороликам, сопровождающим этот урок, создайте основу Вашего проекта "Солнечная система".

Добавьте, по крайней мере, три дополнительные планеты, такие как Меркурий, Венера и Марс, в комплекте с материалами, поведением и аудио.

Отрегулируйте положение планет так, чтобы они представляли собой простую (частичную) модель Солнечной системы.

При необходимости настройте две камеры и кометный спаунер, чтобы все выглядело так, как Вы хотите.

По желанию, не стесняйтесь вносить другие изменения в свой проект, например, добавлять дополнительные объекты в сцену. Это не обязательно, но не стесняйтесь проявлять творческий подход, веселиться, подгонять себя, выделять свой проект и впечатлять своих товарищей!

В идеале создайте сборки проекта для Windows, MacOS и WebGL (но не менее 2 из 3). ПРИМЕЧАНИЕ: Для WebGL убедитесь, что перед сборкой WebGL выбрана опция "Decompression Fallback" в разделе "Publishing Settings" в "Player Settings". Некоторые конфигурации компьютеров испытывают трудности при создании WebGL, особенно если Вы используете неанглийские символы в структуре файлов. В этом случае достаточно будет сборки для Windows и MacOS.

Проверьте, работает ли Ваша сборка. ПРИМЕЧАНИЕ: В идеале Вы можете тестировать на всех платформах, но Вам следует протестировать хотя бы одну (Windows или MacOS). ПРИМЕЧАНИЕ: Локальное тестирование игр на WebGL затруднено из-за особенностей безопасности браузера. Вы можете протестировать WebGL после того, как отправите сборку на сайт itch.io, описанный ниже.

Создайте сжатый архив ZIP с каждой из папок сборки. Чтобы создать zip-архив в Windows, Вы можете щелкнуть правой кнопкой мыши на папке, затем выбрать "Send To->Compressed (zipped) Folder" из контекстного меню. Чтобы создать zip-архив в MacOS, Вы можете щелкнуть правой кнопкой мыши на папке, а затем выбрать "Сжать" из контекстного меню.

Создайте страницу на сайте itch.io для своей игры.

Включите в нее скриншот иерархии Вашей сцены с открытыми всеми родительскими игровыми объектами, чтобы показать организацию Вашей сцены. Если Вам нужны инструкции о том, как делать скриншоты, ознакомьтесь с последующим материалом.

Включите скриншот активов Вашего проекта с открытыми всеми папками, чтобы показать организацию активов.

Включите сборки, созданные выше. Для сборок для Windows и Mac убедитесь, что Вы выбрали "Executable" и выбрали соответствующую платформу. Для WebGL убедитесь, что Вы выбрали "Этот файл будет воспроизводиться в браузере". 

Включите описание произведенных модификаций И титры Вашей игры ("Created by ...").

По желанию включите изображение обложки.

Установите для параметра "Статус релиза" значение "Выпущено".

Установите для параметра "Цена" значение "Без оплаты"

Сохраните страницу и нажмите "Просмотреть страницу"

Убедитесь, что скриншоты отображаются, нажав "Редактировать тему" и установив для параметра "Скриншоты" значение "Боковая панель", и сохраните.

Снова нажмите "Редактировать игру" и установите для параметра "Видимость" значение "Публичный". Вы также можете установить флажок "Unlisted in search & browse".

Внесите любые другие желаемые изменения.

Сохраните страницу и затем нажмите кнопку "Просмотр страницы", чтобы показать то, что увидят Ваши игроки.

Отправьте URL-адрес Вашей страницы игры на itch.io в задание для экспертной оценки. Вы несете ответственность за правильность отправки в соответствии с этими инструкциями. По возможности, проверьте свою отправку на другом компьютере.

Windows
Windows 10 поставляется с инструментами "Snip & Sketch" и "Snipping Tool" для создания скриншотов. В качестве альтернативы Вы можете использовать следующие сочетания клавиш:

Нажмите PrtScn = Это копирует весь экран в буфер обмена. Вы можете вставить снимок экрана в любую программу, которая отображает изображения, например, Paint или Microsoft Word.

Нажмите Alt + PrtScn = Это копирует активное окно в буфер обмена, которое Вы можете вставить в другую программу. 

Нажмите клавишу Windows + Shift + S = Экран затемнится, а указатель мыши изменится. Вы можете перетащить его, чтобы выбрать часть экрана для захвата. Снимок экрана будет скопирован в буфер обмена, который Вы можете вставить в другую программу. (Этот ярлык работает только в том случае, если у Вас установлена последняя версия Windows 10, называемая Windows 10 Creators Update)

Нажмите клавишу Windows + PrtScn = Это сохранит весь экран в виде файла изображения. Вы можете найти его в папке "Pictures", в подпапке под названием "Screenshots"

Существует также множество сторонних инструментов для захвата экрана, например, SnagIt от Techsmith и Camtasia.
